In the examples, assume a student who received 12/15 on “Assignment 1” (which has short name “A1” and is worth 20% of the final course mark), 18/20 on “Assignment 2” (short name “A2”), and 16/20 on “Assignment 3” (“A3”).

Using Formulas

Basic Arithmetic

The +, -, *, and / operators are used for basic arithmetic are you might expect. Parentheses can also be used.

  • 7 * -2 → -14
  • 8+3/2 → 9.5
  • (8+3)/2 → 5.5
Activity Values

You can refer to a student's grade for an activity by putting its name (or short name) in square brackets.

  • [Assignment 1]/10 → 1.2
  • ([A1] + [A2])/3*20 → 200
  • [A1]/[Assignment 2]*3 → 2
  • -[A1] + [Assignment 3] → 4
Activity Properties

You can also refer to an activity's maximum grade or percentage by appending .max or .percent. The .final property of an activity is that activity's contribution to the final percentage grade. That is for an activity A, [A.final] is a shorthand for [A]/[A.max]*[A.percent].

  • [Assignment 1.max] → 15
  • [Assignment 2]/[Assignment 2.max] → 0.9
  • [Assignment 1.percent] → 20
  • [Assignment 1]/[Assignment 1.max]*[Assignment 1.percent] → 16
  • [Assignment 1.final] → 16
SUM Function

The SUM function can be used to add a series of values.

  • SUM(1,2,3,4,5) → 15
  • SUM([A1], [A2], [A3]) → 46
  • SUM([A3],14)/5 → 6
AVG Function

The AVG function can be used to find the (arithmetic) average of a series of values.

  • AVG(1,2,3,4,5) → 3
  • AVG(5,4,3,2,1) + 4 → 7
  • AVG([Assignment 1], [Assignment 2]) → 15
MAX Function

The MAX function can be used to find the largest of a series of values.

  • MAX(1,2,5,4,3) → 3
  • MAX([A1],[A2],[A3]) → 18
MIN Function

The MIN function can be used to find the smallest of a series of values.

BEST Function

The BEST function is used to find the sum of the best n values from a series. The number of values to use, n is given as the first argument; the values are given as the remaining arguments.

  • BEST(3, 6,7,8,9,10) → 27 (The best three values are 8, 9, and 10 and 8+9+10 = 27.)
  • BEST(2, [A1], [A2], [A3]) → 34 (The best two values are 16 (from A3) and 18 (from A2).)

Common Formula Patterns

The most common formula is simply a sum of percentage contributions of each activity to the student's final mark:

  • [A1.final] + [A2.final] + [A3.final] + [MT.final] + [Exam.final]
