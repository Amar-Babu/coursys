    This system is intended to handle as much of the record-keeping and management of a course as possible. Described below are some of the major features of the system.

    Activities and Grades

    Each assignment, exam, etc. is an activity in the system. Or if you prefer, each activity corresponds to a column on your grade sheet.

    Students can be assigned a grade for each activity. (Or grades can be assigned for groups, as described under “Groups” below.)

    Instructors should create an activity for each piece of work in the course. They can also (optionally) use submission and marking tools for each activity.

    Assignment Submission

    The submission feature allows instructors to configure the type of files they want submitted for an assignment.

    Students can then submit their work: files are checked to make sure they are the correct type and meet other criteria set by the instructor. Students can review their submitted files and resubmit if they wish.

    See the assignment submission documentation for more information.

    Assignment Marking

    The “marking” features allow the instructor (or TAs) to set up a marking scheme (or “rubric”) for an activity.

    Each part of the activity will be a component of the marking scheme. For each component, the instructor (or TA) will give a description of what is expected and a maximum mark.

    Then the TA (or instructor) can mark each student against that marking scheme, giving a mark and comment on each component.

    This is used to calculate the student's mark on the activity. When grades are released, students will be able to see the marking scheme, with their marks and comments as feedback on their work.

    Groups

    All of the above can be based on either individual work or group work. If a “group” activity is created, student will be able to form groups for each activity (or for keep groups across multiple activities if desired). Instructors and TAs can manage groups as well, adding and removing members if necessary.

    Student can then submit and receive marks as a group.
