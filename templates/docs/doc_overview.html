    This system is intended to handle as much of the record-keeping and management of a course as possible. Described below are some of the major features of the system.

    Activities and Grades

    Each assignment, exam, etc. is an activity in the system. Or if you prefer, each activity corresponds to a column on your grade sheet.

    Students can be assigned a grade for each activity. (Or grades can be assigned for groups, as described under “Groups” below.)

    Instructors should create an activity for each piece of work in the course. They can also (optionally) use submission and marking tools for each activity.

    Assignment Submission

    The submission feature allows instructors to configure the type of files they want submitted for an assignment.

    Students can then submit their work: files are check to make sure they are the correct type and meet other criteria set by the instructor. Students can review their submitted files and resubmit if they wish.

    See the assignment submission documentation for more information.
