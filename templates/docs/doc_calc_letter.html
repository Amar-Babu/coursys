    Letter grades can be calculated in a calculated letter activity. To create one, select “Add Activity” in your course info page, and select the “Calculated Letter Activity” activity type.

    The Source Activity

    This activity is the one that is used to decide on letter grades. It will likely be the final percentage for your course, either calculated by the system or imported from a spreadsheet.

    Grade Cutoffs

    You can enter grade cutoffs: the lower-bound for each letter grade. When you calculate, each student will have their grade calculated based on their grade on the course activity, and these cutoffs.

    The Exam Activity

    You can optionally set an exam activity for the calculated letter grade. This should likely be the final exam grade for the course. This value is used to guess at the special cases for SFU's final grade values.

    The logic is this:

    1. If the student has no grade for the exam, they will be given an N (incomplete).
    2. If the student has an “excused” grade for the exam, they will be given an DE (deferred exam).
    3. Otherwise, they will have their letter grade calculated based on the source activity as usual.
