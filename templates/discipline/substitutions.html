The following substitutions will be made within the text:

  • {% templatetag openvariable %}FNAME{% templatetag closevariable %}: The student's first name, like “John”.
  • {% templatetag openvariable %}LNAME{% templatetag closevariable %}: The student's last name, like “Smith”.
  • {% templatetag openvariable %}COURSE{% templatetag closevariable %}: The course this case applies to, like “CMPT 001”.
  • {% templatetag openvariable %}SEMESTER{% templatetag closevariable %}: The semester of this course offering, like “Fall {% now "Y" %}”.
  • {% templatetag openvariable %}ACTIVITIES{% templatetag closevariable %}: The list of related activities, like “Assignment 1 and Assignment 2”.

Textile is a simple markup language that you can use to do some basic formatting within this field. Text within a paragraph:

  • _text_: emphasized text (italic)
  • *text*: strongly emphasized text (bold)
  • @text@: code (monospaced font)

And some basic blocks. Each of these should be a separate line/paragraph:

  • bq. text: blockquote (indented as a long-form quote)
  • * text: a bulleted list
  • # text: a numbered list

For more options, see the Wikipedia article on Textile.
